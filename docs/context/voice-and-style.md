# Voice and style

## The cardinal rule

**Preserve Kelly's voice.** Start from her exact words. Fix grammar only. Never AI-polish.

This applies to all prose work in this repo: article bodies, deks, captions, alt text, sidebar copy. The standard is set by the user-level memory at `~/.claude/projects/-Users-kelly-learning-learning/memory/feedback_preserve_voice.md` and the rubric at `docs/rubrics/voice-preservation.md`.

## What "fix grammar only" means

| Always fix | Always preserve |
|---|---|
| Subject-verb agreement | Sentence fragments she chose |
| Typos (`souds` → `sounded`) | Unusual rhythms (`So true. So crazy.`) |
| Capitalization of proper nouns (`apple` → `Apple`) | Period-as-emphasis (`Practice. To. urge.`) |
| Pluralization mistakes | Hedging tics ("honestly," "kinda") |
| Tense agreement when clearly wrong | Casual phrasing she clearly intended |
| Article use (`a AI` → `an AI`) | Doubled words she used for effect (`unless unless`) |

## What "never AI-polish" means

Don't:

- Symmetrize parallel clauses she left asymmetric
- Smooth her uneven rhythm into uniform sentence lengths
- Replace her colloquial phrasing with essayistic equivalents
- Add metaphors, openings, or "bolder rephrasings" unprompted
- Insert design-essay-genre closers (`languages change, frameworks change…` style)

## The read-aloud test

If you read your suggested line aloud, would it sound like Kelly? If not, you've overreached.

## Conventions she uses

- "honestly" / "kinda" / "so true" as conversational hedges — keep
- Em-dashes for conversational pivots — keep
- Period-emphasis (`Practice. To. urge.`) — preserve exactly, including capitalization choices
- Curly quotes throughout (`"text"` not `"text"`) — match this when adding new text
- En-dash for "human–AI interaction" (not hyphen) — match this convention

## When in doubt

Quote her back. Reflecting her own words is almost always more useful than offering yours. If a sentence works, don't rephrase it — even if you could make it "better." Better-sounding is often blander.
