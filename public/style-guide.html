Design System  ·  Apple Human Interface Guidelines

Overview

This design system follows Apple's Human Interface Guidelines. Uses SF Pro (Apple's system typeface) for all text, with JetBrains Mono for code.

Typography

Eight official type styles from Apple Human Interface Guidelines. All use SF Pro with semantic naming.

Large Title
SF Pro · 400
34px / 1.206
Apple HIG: Used for main screen headers and primary titles. The largest standard type size for commanding visual hierarchy.
Large

28px / 1.214
Apple HIG: Primary section headers. Used for important labeled sections within a screen or the titles of modal dialogs and alerts.

Headline
SF Pro · 600
17px / 1.294
Apple HIG: Tappable labels and section headers. The semi-bold weight (600) gives visual emphasis without increasing size. Use for control labels and important text.
Emphasis
Body
SF Pro · 400
17px / 1.294
Apple HIG: Primary body text in lists, paragraphs, and content areas. Default reading size. Most text should be this weight and size. Readable at arm's length without strain.
Default
Subheadline
SF Pro · 400
15px / 1.333
Apple HIG: Secondary labels and supporting text. Smaller than body but still readable. Use for descriptive text below primary labels and in smaller containers.

Callout

16px / 1.313
Apple HIG: Emphasized body text in callout boxes, sidebars, and explanatory notes. Slightly smaller than body (16px vs 17px) but maintains excellent readability in bounded containers.
