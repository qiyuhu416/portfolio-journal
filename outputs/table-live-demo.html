Apple HIG Specifications

Header Font
SF Pro (system font stack) — Subheadline: 15px, 400 weight, 1.333 line-height
Body Font
SF Pro (system font stack) — Body: 17px, 400 weight, 1.294 line-height
Padding
Headers: 18px 24px | Cells: 20px 24px (following 8pt grid)
Borders
1px hairline separators (var(--line)) + 12px border radius
Visual Enhancement
Subtle alternate row backgrounds (2% opacity) for improved readability
Spacing
48px top/bottom margins for proper breathing room in articles
Header Background
Subtle surface color (var(--surface)) for visual separation
Contrast
Headers: ink-2 (secondary) | Body: ink (primary) — WCAG AA compliant
