Font Families
Primary — SF Pro
Apple's native typeface. Used across iOS, macOS, watchOS, and tvOS. System font stack uses -apple-system to load SF Pro on Apple devices, with Inter as a fallback. 8 official Apple HIG semantic type styles: Large Title, Title, Headline, Subheadline, Body, Callout, Caption 1, Caption 2.

Code only. Terminal, inline code, code blocks.

Figure
Single image, video, or placeholder. Supports multiple widths via span prop.
Props: src, alt, fig (label), caption, span ('text'|'full'|'bleed'), height, placeholder
Gallery
Grid container for multiple Figures. Simple wrapper, 2–3 columns.
Props: children, columns (2|3, default 2)
Carousel
Horizontal scrolling carousel with prev/next controls and page dots. Syncs with manual scroll/swipe.
Props: items (array of {src, alt, caption}), perView (default 2), span ('text'|'full'|'bleed', default 'full')
Collage
Expandable moodboard. Two layouts: compact card-stack preview (default, click to expand), or full-width tilted strip.
Props: items (array with src, alt, caption, aspectRatio), caption, layout ('preview'|'strip', default 'preview'), span, defaultExpanded
TabbedFigure
Tab interface that switches between labeled figures. Each tab can have an abstract SVG preview (click to reveal full image), custom figure, or image URL.
Props: items (array with label, src/figure/abstract, alt, caption, when, where), fig (caption), span ('text'|'full'|'bleed', default 'text')
Constraints
No coloured card borders — colour goes in the label only Cards
At most one label inside any non-content component Components
No inline font-size or margin overrides inside Sidebar — CSS normalises list spacing Sidebar
Serif (Switzer) for all editorial text. Sans (Inter) for labels, nav, meta only. Type
Uppercase labels always fontWeight 500 — 400 is too thin at 12px all-caps Type
Tints only on interaction — never body text or decorative backgrounds Colour
All font sizes minimum 12px (WCAG AA) — no sub-12px anywhere in the site Accessibility
All text colours pass 4.5:1 on --bg — --ink-3 (#6B6560) is the lightest text value Accessibility
