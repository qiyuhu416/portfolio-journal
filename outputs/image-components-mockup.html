Image Components

Five reusable components for displaying images, videos, and interactive media.

Figure

Single image, video, or placeholder
The foundational image component. Renders `<img>`, `<video>` (autoplay loop), or placeholder. Supports three width spans.

Carousel

Directional carousel with controls
Horizontal scrolling strip. Shows N items at once. Includes prev/next arrow buttons + page indicator dots. Syncs with manual scroll/swipe.

Collage

Expandable moodboard or tilted strip
Two layouts: (1) Compact preview — three cards fanned like a deck, expand on click to horizontal scrolling row. (2) Strip — full-width tilted row, always expanded, bleeds beyond the column.

TabbedFigure

Tab switcher between labeled figures
Tab interface at top. Each tab can have: (1) Abstract SVG preview (collapsed, click to reveal full image), (2) Custom rendered SVG, or (3) Image URL. Constant aspect ratio prevents layout shift.

Usage notes

Span prop (Figure, Carousel, TabbedFigure):
• span="text": 680px max-width (fits article column, default)
• span="full": 1040px max-width (room for embedded text/labels)
• span="bleed": 100% width (full viewport, atmospheric)

All images:
• Auto-rounded corners (8px border-radius)
• Respect article tint + surface tokens for color consistency
• Support placeholder rendering when src is omitted
• Captions use --sans 12px (mono-style labels)

Accessibility:
• All require alt text (screen readers, fallbacks)
• Placeholders show FIG. N label for sighted users
• Carousels + Collage + TabbedFigure are keyboard navigable
• All pass WCAG AA for color contrast
